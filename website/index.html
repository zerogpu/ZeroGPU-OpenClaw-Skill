Manual setup (if you prefer)

  1. Run `openclaw plugins install zerogpu-router`.
  2. Run the setup script to store ZeroGPU credentials.
  3. Enter the user's ZeroGPU API key and project ID when prompted.
  4. Keep the existing primary model and let the agent offload small tasks to ZeroGPU tools.
  5. Confirm dashboard events on the hosted adapter after a live call.

Frequently asked questions

Can I call ZeroGPU directly without this tool?

Yes. The plugin is for routing, normalization, and savings tracking on top of direct API usage.

Do users need to host anything?

No. Users only configure OpenCLAW with a hosted adapter URL and their own ZeroGPU credentials.

How do I verify savings?

Use `/dashboard/events` and `/dashboard/summary`. Each response also includes `zerogpu` metadata.
